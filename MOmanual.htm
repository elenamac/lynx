SYSTEM OPERATOR MANUAL

Study this manual carefully; you are the expert. In these pages you will find everything you need to know how to fix any problems with your submersible. And be careful — one small mistake and you could have a failed mission or a lost rover on your hands.
Problem solving
The submersible will be destroyed when its countdown timer reaches 0:00 or when too many strikes (mistakes) have been recorded. The only way to keep your submersible working and on its mission is to correctly respond to all the rover’s modules before the countdown timer expires.


Modules
Each submersible will include several modules that you must respond to, adjust, or fix. You can work with the modules in any order. Instructions for working with the modules can be found in Sections 1 and 2.

Strikes
Mistakes made by the System Operator can result in damage to the rover. Damage is tracked on the control panel with a strike (“X”) on the indicator above the countdown timer. Submersibles with a strike indicator will fail their mission, be lost or destroyed.


Gathering Information
Some instructions will require specific information about the rover, such as the serial number. This type of information can be found on the top, bottom, or sides of the control panel casing. Each modules has an LED in the top right corner. When this LED is lit green the module has been properly responded to or fixed. All modules must be responded to or fixed in order to complete the rover’s mission successfully.

THE WIRE MODULE
A wire module can have 3-6 wires on it. In all cases, either a single wire will need to be cut or a fuse will need to be fixed in order to successfully complete the module. Wires are ordered with the first wire at the top.
When there are 3 FUSES:
  1. If there are NO BLUE fuses, then cut the third wire on the left side.
  2. Otherwise, if the last fuse is RED, cut the third wire on the left side .
  3. Otherwise, if two or more fuses are YELLOW, cut the second wire on the right.
  4. Otherwise, cut the first wire on the left.

When there are 4 FUSES:
  1. If there are 2 or more BLUE wires and the engine model number is larger than 2000, cut the first wire on the right side.
  2. Otherwise, if the first fuse is PURPLE and there are no BLUE wires, cut the second wire on the left side.
  3. Otherwise, if there is exactly one RED fuse, cut the left and right hand wires of the last fuse.
  4. Otherwise, if there is more than one YELLOW fuse, cut the left hand wire of the second fuse.
  5. Otherwise, cut the wire to the right of the third fuse.
  6. If there are 2 or more BLUE wires and the engine model number is larger than 2000, cut the first wire on the right side.
  7. Otherwise, if the first fuse is PURPLE and there are no BLUE wires, cut the second wire on the left side.
  8. Otherwise, if there is exactly one RED fuse, cut the left and right hand wires of the last fuse.
  9. Otherwise, if there is more than one YELLOW fuse, cut the left hand wire of the second fuse.
  10. Otherwise, cut the wire to the right of the third fuse.

When there are 5 FUSES:
  1. If the second fuse is WHITE and the Model number is less than 2000, cut the right hand wire for the last fuse.
  2. Otherwise, if there is more than one RED fuse and more than one YELLOW fuses, cut the wire to the left of the 3rd fuse.
  3. Otherwise if there are no PURPLE fuses, cut the wire to the right of the third fuse.
  4. Otherwise, cut the wire to the left of the second fuse.
When there are 6 FUSES:
  1. If the model number is more than 2010, and there are no WHITE wires, cut the wire to the left of the sixth fuse.
  2. Otherwise, if there is more than one BLUE fuse and less than 3 RED fuses, cut the fourth wire to the right of the fuse panel.
  3. Otherwise, is there are no YELLOW wires and more than one WHITE wire, cut the second wire to the left of the fuse panel.
  4. Otherwise, cut the wire to the left of the sixth fuse.

THE BUTTON MODULE
If the instruction below does not apply, hold down the button and see the “Next Step Instructions.”

  1. If the button is red, hold down the button and see the “Next Step Instructions.”
  2. If the gimbal indicator is green and the button is blue, quickly press and release the button.
  3. If the gimbal indicator is red and the button is white, hold down the button and see the “Next Step Instructions.”
  4. If the amp meter reads more than 50 Amps and the button is grey, quickly press and release the button.
  5. If the button is green, hold down the button and see the “Next Step Instructions.”
  6. If the button is yellow and the gimbal indicator is white, quickly press and then release the button.
Next Step Instructions
  1. If the button is red, release the button when the Amp meter reads less than 10 Amps.
  2. If the button is white, the gimbal indicator is red, and the button text says ”Safety” release the button when the Amp meter is less then 5 Amps.
  3. If the button is green, and the text above it says “alert” hold down the button and release it when the Amp meter is less than 10 Amps.
  4. Otherwise, release the button when the Amp meter is at an even number.
THE BALLAST PUMP MODULE
There are four ballast tanks that hold ballast water. There are separate pumps for each tank. Activation of the pumps needs to be coordinated. Under different conditions, water from some tanks needs to be pumped first while water from other tanks is pumped last.
  • One of the tank numbers will flash.
  • Using the Tank-Pump directions below, activate the correct pump.
  • The original tank will flash, followed by another. Repeat this sequence in order using the Tank-Pump directions below.
  • The sequence will lengthen by one tank each time you correctly enter a sequence until the rover’s correct buoyancy is achieved.

THE NAVIGATION MODULE
  • Find the map with the matching yellow buoys.
  • The system operator must navigate the submersible to the ship using the arrow buttons on the control panel.
  • Warning: Make sure the system operator does not run into any rocky outcrops. This will cause damage to the submersible.
  • The rocky outcrops can not be clearly seen by the system operator.
